# /sprint-plan

Stwórz plan sprintu (1-2 tygodnie) dla ClawCard.

## Kroki
1. Przeczytaj docs/SESSION_STATE.md — co jest zrobione
2. Przeczytaj docs/TASKS.md — co czeka
3. Przeczytaj docs/BUGS.md — otwarte bugi
4. Zaproponuj 3-5 zadań na sprint z priorytetami
5. Zapisz plan do docs/SESSION_STATE.md sekcja "Sprint Plan"
6. Zrób git commit: "docs: sprint plan YYYY-MM-DD"

## Format
```
## Sprint [data_start] → [data_end]

### Cel sprintu (jedno zdanie)
...

### Zadania
- [ ] P1: [zadanie] — [agent odpowiedzialny] — est. [czas]
- [ ] P2: [zadanie] — [agent odpowiedzialny] — est. [czas]
- [ ] P3: [zadanie] — [agent odpowiedzialny] — est. [czas]

### Definition of Done
[co musi działać żeby sprint był sukcesem]
```
