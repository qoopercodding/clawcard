# ClawCard — Roguelike Deck Builder

Dark fantasy deck builder inspirowany Wildfrost. Stack: React + Vite + TypeScript.
Repo: https://github.com/qoopercodding/clawcard

## Struktura
- wildfrost-poc/clawcard-builder/src/ — kod Builder Tool
- wildfrost-poc/clawcard-builder/src/pages/StartPage.tsx — Start Screen
- wildfrost-poc/clawcard-builder/src/modules/ — moduły (card-editor, frame-editor, map-editor)
- tools/card-scraper/ — scraper kart

## Design system
Ciemny dark fantasy: tlo #1a0e05, akcent #c8902a (zloto), tekst #e8d5b0

git add . && git commit -m "..." && git push
